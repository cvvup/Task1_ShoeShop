# ООО «Обувь»

Учебное приложение на Python и SQLite, выполненное по ТЗ практического задания.

Приложение реализует:

- вход по ролям;
- просмотр товаров;
- поиск, фильтрацию и сортировку товаров;
- добавление, редактирование и удаление товаров;
- просмотр заказов;
- добавление, редактирование и удаление заказов.

## Что находится в проекте

- `prj.py` — основной файл приложения
- `schema.sql` — SQL-скрипт структуры базы данных
- `shoe_store.db` — рабочая база данных SQLite
- `resources/import` — исходные файлы заказчика для импорта
- `resources/product_images` — изображения, которые использует приложение
- `.pylintrc` — настройки линтера `pylint`
- `requirements-dev.txt` — зависимости для линтера

## Роли в приложении

- `Гость` — может только смотреть товары
- `Авторизированный клиент` — может смотреть товары
- `Менеджер` — может смотреть товары, искать, фильтровать, сортировать и смотреть заказы
- `Администратор` — может работать с товарами и заказами полностью

## Как запустить

Обычный запуск:

```bash
python3 prj.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:8000/login
```

## Если нужно вернуть базу к начальному состоянию

Эта команда удаляет текущую базу, создаёт её заново и повторно загружает данные из папки `resources/import`:

```bash
python3 prj.py --init-db
```

Если нужно только создать базу без запуска сайта:

```bash
python3 prj.py --init-db --no-server
```

Можно в любой момент восстановить исходные данные (например, удалить все позиции и восстановить без потерь).

## Как работает база данных

- структура базы описана в `schema.sql`
- при инициализации создаётся файл `shoe_store.db`
- данные из Excel-файлов из `resources/import` загружаются в эту базу

## Линтер

Для проверки Python-кода используется `pylint`.

Если нужно установить линтер:

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements-dev.txt
```

После этого в VS Code можно использовать подсветку замечаний линтера.
